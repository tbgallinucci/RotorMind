# knowledge/raw

Source documents for the wiki live here (the wiki-ingest workflow mirrors this
tree into `assistant/wiki/index.md`).

The seed corpus shipped with this repo was written directly as wiki pages -
original summary notes citing Childs (1993), Vance (1988), Reynolds (1886,
public domain) and API 684 concepts *in the author's own words*. No standard
or book text is redistributed. Drop your own PDFs/notes here to extend the
library; never commit documents you don't have the right to redistribute.
